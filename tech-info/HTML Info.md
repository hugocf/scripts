## abbreviation

```html
<abbr title="title">abbreviation</abbr>
```



## anchor / link

```html
<a href="">hyperlink</a>
```



## blockquote

```html
<blockquote>
</blockquote>
```

```html
<blockquote>
    <p>Quote.</p>
    <footer>— <cite class="author">Author</cite> (in <cite class="title">Source</cite>)</footer>
</blockquote>
```

*(in Markdown, via [StackOverflow](https://stackoverflow.com/a/2002150))*

```markdown
> Quote.
>
> — <cite>Author</cite>
```
